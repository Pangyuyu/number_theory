# 冰雹猜想

## 定义

>取任意一个正整数n，若n为奇数，则下一步变成3n+1；若n为偶数，则下一步变成n/2。  
>无论n是怎样一个数字，最终都无法逃脱回到谷底1。准确地说，是无法逃出落入底部的4-2-1循环，永远也逃不出这样的宿命。这便是著名的“冰雹猜想（3N+1猜想）”
